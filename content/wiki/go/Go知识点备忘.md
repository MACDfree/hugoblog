# =和:=的区别

=用于赋值，:=声明变量并赋值且进行自动类型推断
=必须先使用var进行声明

``` go
var a
a=100
// 或
var b=100
// 或
var c int=100
// :=声明并赋值，不能加var，也不能加类型
d:=100
```

# slice类型和array类型变量初始化差异

数组：[n]T

切片：[]T

``` go
s := []int{0,1,2,3,4,5} // slice
s := [...]int{0,1,2,3,4,5} // array
values := [9]int{} // array
```

切片初始化时不需要指定长度，数组初始化时需要指定长度
